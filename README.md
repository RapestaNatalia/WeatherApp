# WeatherApp

Steps to execute application:

# Client:
    -cd weatherApp
    -npm install
    -npx react-native run-android

# Server:
    -cd server
    -npm install
    -Create .env file inside /server
    -Set the values:
        PORT=
        WEATHER_URL= ''
        WEATHER_KEY= ''
        IP_API_URL=''
        IP_API_KEY=''
    -npm start
